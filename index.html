Bất Hòa Bot

Bot Discord hỗ trợ quản lý máy chủ, ticket, phòng thoại tạm thời và các tiện ích khác.

Thông tin

Bot có thể truy cập một số thông tin Discord cần thiết để hoạt động như:

  • ID người dùng
  • Tên hiển thị
  • ID máy chủ
  • ID kênh
  • ID vai trò
  • Cấu hình bot trên máy chủ
